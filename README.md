## 🎮 How to Use

After installation, the extension integrates directly into After Effects. Follow these steps to load and run a script:

1. Open Adobe After Effects and your project.
2. Navigate to the **Window** menu → **Extensions** → **AEScripts Exte**.
3. In the extension panel, click **Load Script** and select your `.jsx` or `.jsxbin` file.
4. Configure any script parameters in the UI, then click **Run**.
